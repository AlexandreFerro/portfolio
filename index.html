Curso Superior de Análise de Sistemas

Projetos interdisciplinares:

1ª Semestre: Foi criada uma empresa fictícia com ênfase em sustentabiidade, com o intuito de resgatar equipamentos e softwares antigos, dados como obsoletos, na intenção de ampliar o mercado para aqueles com poucas condições de informatizar seu negócio. Criamos um sistema de controle de estoque que fazia cadastro, compra e venda de serviços. Criamos a identidade visual e a planta da empresa em Visio. A programação foi em VBA e o Banco de dados utilizado foi o Access.

2ª Semestre: Neste semestre demos seguimento a ideia inicial, imergindo desta vez em disciplinas como Engenharia de Software e entendendo novas ferramentas como Function Point, Diagrama de caso de uso, Testes da caixa branca/preta e PDCA. Contamos também com o MS Project para o gerenciamento e escala do projeto. A linguagem desta vez foi o C#, o Banco de dados MySql e a ferramenta de testes Selenium.

3ºSemestre: Mantivemos o tema ainda por se tratar de algo que realmente acreditávamos, conscientizar a todos de que hoje em dia a sustentabilidade é de fundamental importância. Além disso, achamos que preservar o escopo e alterar/acrescentar novas tecnologias, ferramentas a um projeto já existente seria mais fácil e menos arriscado do que partir para uma idéia totalmente nova. Desta vez nos foram apresentadas ferramentas de cálculo, como o TCO (Total Costs of Ownership) e o ROI (Return on Investiment).Utilizamos também o software livre Draw.io para a confecção esquemática. A linguagem de programação foi Java e o Banco de dados o MySQL, da qual criamos a conexão. Demos inÍcio ao estudo de POO neste semestre.

4ºSemestre: Neste projeto resolvemos mudar totalmente o tema, de modo a dar um refresh ao curso e experimentar recursos diferentes, no sentido de não ficar só num produto para controlar estoque. Além disso partimos para um campo que, por acaso, também era do agrado de todos, como a música. Desta vez desenvolvemos uma plataforma de degustação, compra e venda de samplers. Passamos do projeto em papel para o Adobe XD que permite toda a elaboração do protótipo do software, inclusive a interação das telas. As linguagens foram Python, Django e HTML. O Banco de dados foi o SQL Server. Criamos um video demonstrativo.

5ºSemestre: No PI (Projeto Interdisciplonar) deste semestre criamos uma empresa de venda de pacotes de viagens. Trabalhamos com Programação Mobile. Tivemos uma iniciação no Android Studio sobre as técnicas de interação com alguns celulares. O Banco de Dados foi o nativo SQLite.

6ºSemestre No último semestre, para o trabalho final, desenvolvi uma idéia da qual já trazia desde o início do curso, que era criar oportunidades como oferta de serviços e novos empregos em horário alternativo. Baseado em plataformas bem conhecidas como o iFood e o Get Ninjas, estudei sobre um sistema que oferecesse uma ampla gama de serviços para pessoas que tivessem disponibilidade de prestação de serviços especialmente em momentos não convencionais. Utilizei o HTML, Php e o My SQL para conceber o projeto.
